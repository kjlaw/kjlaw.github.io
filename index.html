Portfolio

Hackathon Projects

Kinetic Connection - An Android app that allows users to “kinetically connect” with people on LinkedIn in augmented reality. Created using a customized camera interface and a face recognition API from Kairos to display an overlay over the person’s face with their name and title, and a button that deep-links into their profile within the native LinkedIn app.

Sm:)e - An Android app that helps users keep their mental health in check by analyzing the tone of recent texts, keeping track of their mood, and promoting auditory and visual relaxation. Utilized Android's Telephony provider and IBM Watson's Tone Analyzer to obtain emotional, writing, and social tones for the user's texts.

Wayst - A haptic feedback waist accessory that provides navigation assistance via proximity sensors and a companion Android app. Android app utilizes the Google Maps Directions API and the Particle Android Cloud SDK to provide navigation information to the Particle Core in the belt.

UberPrepared - An Uber pre-order/scheduling service that works via SMS or a native Android app. Developed with the Uber, Twilio, and Google Places APIs, a finite state machine, and some natural language processing techniques to allow users to fuzzy search pick-up and drop-off locations and select a pick-up time. Implemented the backend using Node.js, ExpressJS, and MongoDB.

Wake Me Up - A Pebble app, developed in C, and a companion Android app that wakes a user from their nap while taking public transportation. Uses Google Play services location APIs to track the user’s location and wake them before they reach their stop.

OZBOT - OZBOT is a web app developed using HTML, CSS, and JavaScript that takes young girls through the story of The Wizard of Oz and teaches them about various aspects of robotics along the way. The end of the current app is a springboard for our next project, where we hope to turn "Oz" into a digital workspace that allows simple robot assembly and very basic CAD and programming exposure. This workspace would enable girls to design their own robots and give them the option to purchase their custom kit of parts to build the robot at home


Personal Projects

DrawIt - An iOS app that allows users to take a picture, draw or write a message on it, and share with friends. Used a variety of Swift APIs, including Core Data, which saves the user’s creations in persistent storage for future querying.

Scorekeeper - A native Android app developed using Eclipse to keep track of the score of a ping-pong game. At the end of the game, the serving win percentage of each player is calculated and displayed

Blog Reader - An Android app created in Eclipse to display blog posts from a blog. It gives users an option in the settings to choose which blog to view

Motivational Quotes - A Motivational Quotes Generator Android app developed to play around with animations


Website Projects

Kash Development, LLC - Kash Development, LLC is a family-owned business that provides rental homes in East Lansing, Michigan. The purpose of this website is to make it easy for people to locate the properties and find out information about them. I decided to add all of the properties to a Google map, so potential renters can locate them easily and get directions if necessary. I also created a photo album and an information list for each property. The website also features downloadable lease documents and a conveniently placed phone number in the header.

H.E.A.T. Tennis Academy - H.E.A.T. Tennis Academy offers summer camps and practice sessions for tennis players of all levels. The purpose of this website is for players to have a convenient way to access information about the coaching staff, schedule, tuition, location, and announcements. I used an interactive map that displays the academy’s location with current weather information at that location. I also added a Twitter feed to display schedule and location changes due to weather. Using Adobe Flash, I customized a Flash template and added it to the intro page as requested. I also set up social media sites such as Facebook and Twitter for this academy and designed promotional brochures.

FIRST in Michigan - FIRST in Michigan is a non-profit organization that oversees all the FIRST robotics competitions in Michigan. I started working on this website in 2010, adding new components and features over the years. Working closely with the board of directors, I designed the FIRST in Michigan logo based on the US FIRST logo. The color scheme of this website also follows the US FIRST color scheme. This website contains information for all levels of robotics competition, from Kindergarteners building LEGO robots to high school students building robots weighing over 100 pounds. This website displays all the events in Michigan and information relating to these events. It also has an interactive map that displays all the FIRST robotics teams in Michigan.

Destination STEM - Destination STEM is a non-profit organization that offers free summer camps to young girls with the purpose of increasing their interest in Science, Technology, Engineering, and Mathematics (STEM). I designed a logo for this organization, and I selected an appropriate color scheme. This website displays camp information, schedules, and downloadable application forms. Also featured on this website are videos that I created using previous years’ pictures and video clips. In addition, I created a sponsors page to display the sponsors and link to their websites.

Educational website - I created this website for a teacher to make learning about the American government fun for early elementary school students. This website features interactive graphics that “pop-up” to reveal more information about certain topics. Students are also able to download activities and homework assignments from the website. Phillip Martin’s artwork was used extensively to create this site.

Bionic Barons Robotics - I created this website for the Bionic Barons, Andover High School’s robotics team. The main purpose of this website is to generate interest in the community and to promote STEM (Science, Technology, Engineering, and Mathematics). A medieval theme was chosen for this website since it is the home of the barons. Graphics and animations are used throughout this website to make it appealing to younger kids. Malathar’s dragons were animated using Photoshop, and a customized Google map is also displayed on the website. prettyPhoto was used to display pop up videos on certain pages. The banners on the side feature quick links to the US FIRST robotics website, the FIRST robotics in Michigan website, and the websites of the team’s major sponsors.

Andover High School Golf - I created this website for the Andover High School golf team’s 2011 season. It also featured a photo gallery where visitors could view and download pictures.

Andover High School Tennis - I created this website for the Andover High School tennis team’s 2011 season. Armand Molino Tennis - I built this website in 2009 for Armand Molino, a tennis coach in SE Michigan. Coach Molino wanted a website where he could direct people looking for more information about his background and his summer camps. This website ended up serving multiple purposes. It became a place where campers can go for the schedule, camp news, and announcements. It also displays a Twitter feed containing information about rain delays and last-minute schedule changes. In addition, this website is host to Molino’s summer camp photo archives where campers can view previous years’ camp pictures, and a hall of fame displaying the names of his students who were tournament winners. I also worked with Coach Molino to design camp brochures and t-shirt designs for his summer camps.
